# ProyectoGestion

Sistema de gestión de proyectos de software en Java (mini Jira).

## Descripción

Este proyecto permite crear proyectos, agregar tareas, cambiar estados y eliminar tareas desde la consola. Es un gestor simple para organizar tareas de software.

## Funcionalidades

- Crear proyectos
- Seleccionar un proyecto
- Agregar tareas
- Cambiar estado de tarea: pendiente, en progreso, terminado
- Mostrar lista de tareas
- Eliminar tareas

## Requisitos

- Java JDK instalado

## Ejecución

Abre la terminal en la carpeta `src` y ejecuta:

```bash
javac Main.java Proyecto.java Tarea.java
java Main
